Cross-Site Scripting (XSS) in User Profile

User-provided biography text is rendered without escaping, allowing malicious scripts to execute in other users' browsers.

Impact:

Account takeover, data theft, malicious actions performed on behalf of users

// Recommended fix:

// 1. Use React's default JSX escaping

// 2. Or sanitize with DOMPurify if HTML is needed
